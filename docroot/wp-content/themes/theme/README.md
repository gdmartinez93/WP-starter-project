# Starter theme

Features:

- Webpack 4.4
- Gridlex
- Swiper
- Hamburguer CSS

## Setup instructions

1. Clone this repo into your Projects directory

    ```
    git clone https://github.com/gdmartinez93/starter-theme YOUR_PROJECT
    cd YOUR_PROJECT
    ```

1. Initialize the theme

    This will install the dependencies

    ```
    npm install
    ```
